Pagina administrativa de una Biblioteca

Informacion del proyecto

  • Categoria: Aplicacion Web
  • Fecha de proyecto: 14 Enero, 2022
  • URL del repositorio: https://github.com

Este proyecto, fue desarrollando con ASP.Net Core MVC, usando como ORM Entity Framework Core con arquitectuta por capas, la base de datos, fue creada En SQL Server, es un proyecto para la administracion de libros vendidos y alquilados.
