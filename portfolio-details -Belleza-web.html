Aplicacion Web administrativa de un Salon de Belleza

Informacion del proyecto

  • Categoria:Aplicacion Web
  • Fecha del proyecto: 23 Julio, 2021
  • URL del repositorio: https://github.com

Este proyecto se creo con, ASP.Net Core MVC, usando como ORM Entity Framework Core con arquitectuta por capas, para la creacion del Frond-End se uso Bootstrap , la base de datos, fue creada En SQL Server, es un proyecto para la administracion de un salon de belleza, la funcion principla de este programa es adminitrar, las citas, los empleados, y los servicios que ofrece un salon de belleza.
